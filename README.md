# FormComponents

A collection of reusable React form components built with Material UI.

## Installation

Install the required dependencies:

```bash
npm install @mui/material @emotion/react @emotion/styled
```

## Features

- Reusable form components
- Material UI integration
- Responsive design
- Easy customization
- Ready for any form management solution

## Included Components

- Text Field
- Text Area
- Select
- Autocomplete
- Checkbox
- Radio Group
- Switch
- Date Picker

## Date Picker

The Date Picker component uses **react-multi-date-picker** with Persian calendar and Persian locale support.

Additional packages required for Date Picker:

```bash
npm install react-multi-date-picker react-date-object
```


## Technologies

- React
- Material UI
- React Multi Date Picker
